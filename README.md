# system-optimize-magic-cube
## 系统优化魔方
Windows系统底层性能优化专用工具，魔方系列独立本地EXE桌面程序

## 一、核心功能
1. Windows后台服务一键管控，禁用冗余占用进程
2. 系统缓存、日志、垃圾文件深度批量清理
3. NVIDIA显卡算力调度优化，适配RTX3060/3070/3080系列显卡
4. 开机启动项、内存占用可视化调节，降低系统负载
5. 磁盘碎片、分区读写速度优化

## 二、硬件&系统适配
- 系统：Windows 10 / Windows 11 64位
- 推荐配置：i5及以上处理器、16G/32G大内存、N卡独显主机
